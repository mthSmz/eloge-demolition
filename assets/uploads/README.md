# assets/uploads

Déposez ici vos images importées manuellement.

## Convention recommandée

- Illustrations de dossier : `<critique-id>-eloge.<ext>` et `<critique-id>-demolition.<ext>`
- Avatars : `<prenom-nom>-avatar.<ext>`

Exemples :
- `li-po-eloge.jpg`
- `li-po-demolition.jpg`
- `ulysse-poussin-avatar.jpg`
- `achille-corbeau-avatar.jpg`

Formats conseillés : `jpg`, `png`, `webp`.
