Position éditoriale

Manifeste

Un laboratoire critique simple : deux angles, un même texte, zéro dogme.

Pourquoi ce dispositif existe

Éloge / Démolition part d’une idée volontairement sobre : la vérité absolue n’existe pas en critique littéraire. Il n’y a que des lectures situées, argumentées, discutables. Le site met donc en scène deux voix contradictoires pour rappeler qu’une œuvre peut être défendue et contestée sans trahir le texte.

Comment on limite les hallucinations

  1. 1 — Génération structurée

    D’abord un JSON intermédiaire : observations factuelles, citations exactes, puis interprétations ancrées sur ces citations.

  2. 2 — Audit déterministe

    Chaque citation doit exister littéralement dans le texte source. Les entités nommées doivent, elles aussi, être présentes dans le texte.

  3. 3 — Rejet si échec

    Si une règle casse (citation invalide, entité inventée, interprétation non justifiée), la génération est rejetée et non “corrigée” automatiquement.

Message implicite (et assumé)

Ce projet de portfolio n’essaie pas de “faire plus” : il essaie de faire proprement. Définir un cadre, séparer production et validation, documenter les règles, tester les sorties, puis itérer sans gonfler le scope. C’est une démonstration de méthode autant qu’un objet éditorial.
